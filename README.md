# jpreagan.com

- **Framework**: [Next.js](https://nextjs.org/)
- **Deployment**: [Vercel](https://vercel.com)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/

## Running Locally

This application requires Node.js v16+.

```bash
git clone https://github.com/jpreagan/jpreagan.com.git
cd jpreagan.com
yarn install
yarn dev
```
